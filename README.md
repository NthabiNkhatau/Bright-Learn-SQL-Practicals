# Bright-Learn-SQL-Practicals
The repository contains a structured set of SQL solutions for Practical 1: SQL Fundamentals (Databricks – Basic SQL Syntax), built using the retail_sales_dataset.csv and designed to demonstrate a strong foundation in querying structured data within a real-world transactional context . The work systematically covers core SQL concepts including SELECT statements, DISTINCT queries, conditional filtering using WHERE clauses, aggregate functions such as COUNT, AVG, SUM, MAX, and MIN, as well as GROUP BY and HAVING for data aggregation and filtering, and CASE statements for deriving meaningful categorical insights from raw numerical data. Each query is intentionally written to address one concept at a time, in line with the practical’s requirements.
